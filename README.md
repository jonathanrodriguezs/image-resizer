# Image Editor

Python image editor

## Install dependencies

```shell
pip install -r requirements.txt
```

## Run application

```shell
python editor.py
```

## Lint all py files

```shell
sh linter.sh
```

## Create Windows executable

```shell
pyinstaller --noconsole --onefile editor.py
```
